### 🎯 **How to Contribute**

1. **Fork the repository**
   ```bash
   git fork https://github.com/your-username/ai-chatbot-comparison-2025.git
   ```

2. **Create a feature branch**
   ```bash
   git checkout -b feature/model-update
   ```

3. **Make your changes**
   - Update model data in `data/models.json`
   - Add new sources to `docs/REFERENCES.md`
   - Update the main table in `index.html`

4. **Commit with clear messages**
   ```bash
   git commit -m "Update: Add GPT-5 energy consumption data"
   ```

5. **Submit a Pull Request**
   - Provide detailed description of changes
   - Include source verification for new data
   - Reference issue numbers if applicable

### 📋 **Contribution Guidelines**

#### ✅ **We Need Help With:**
- 📊 **Data Updates**: New model releases and specification changes
- 🔍 **Source Verification**: Cross-checking data with additional sources  
- 🌍 **Translations**: Multi-language support for global accessibility
- 🎨 **UI/UX Improvements**: Enhanced visualizations and user experience
- 📱 **Mobile Optimization**: Better responsive design and touch interactions
- 🔧 **New Features**: Interactive filtering, comparison tools, export options

#### 📝 **Data Quality Standards:**
- **Minimum 2 sources** for any new metric
- **Official documentation preferred** for specifications
- **Peer-reviewed sources** for energy consumption data
- **Recent data only** (within 6 months for dynamic metrics)
- **Full citation required** with URL and access date

#### 🚫 **Not Acceptable:**
- Unverified or single-source data
- Marketing claims without independent validation
- Outdated information (>1 year for static specs)
- Proprietary or confidential information
- Biased or promotional content
